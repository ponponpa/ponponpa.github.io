什么是路由？
路由是根据不同的 url 地址展示不同的内容或页面。

React Router
一个针对React而设计的路由解决方案、可以友好的帮你解决React components 到URl之间的同步映射关系。

安装依赖 使用2.8.1版本的依赖

npm install react-router@2.8.1 -D
